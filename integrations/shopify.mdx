## Setting Up Shopify API Keys

Before you can use the Shopify integration, you'll need to set up your API credentials. Here's how to get your API keys:

1. Log in to your [Shopify Partner Dashboard](https://partners.shopify.com)
2. Navigate to Apps > Create App
3. Choose "Create App Manually"
4. Fill in your app name and select the app type (Public or Custom)
5. Once created, go to "API credentials"
6. You'll need these credentials:
   - API Key
   - API Secret Key
   - Access Token (generated when installing the app in a store)
